# 🧠 AI SYSTEM INSTRUCTIONS FOR LLM-GATEWAY-SERVICE
1. **Teknoloji:** Rust 1.84+ (Tokio, Tonic).
2. **Stateless Zorunluluğu:** Servis %100 stateless çalışmak zorundadır. Hiçbir veritabanına (Redis/Postgres) bağlanamaz, durum tutamaz. Sadece proxy/yönlendirme yapar.
3. **mTLS Strict:** gRPC (Tonic) sunucusu ve istemcileri ayağa kalkarken `ca.crt` eksikse `panic!` atıp çıkmalıdır. Güvensiz (Insecure) HTTP fallback kesinlikle YASAKTIR.
4. **Loglama:** Standart `println!` yasaktır. Daima SUTS v4.0 JSON formatında (`tracing` crate ile) log basılmalı ve `x-trace-id`, `x-tenant-id` mutlaka `Span` içine eklenmelidir.
